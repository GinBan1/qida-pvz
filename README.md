# 齐大植物大战僵尸

一个以齐齐哈尔大学为主题的网页版植物大战僵尸同人小游戏，基于 Canvas 与原生 JavaScript，实现关卡挑战与无尽生存，并包含排行榜与昵称系统。

## 功能特色
- 关卡模式（多难度关卡）
- 无尽生存模式
- 阳光收集、植物布置、冷却与铲除机制
- 排行榜与昵称系统（本地存储，支持对接远端存储）
- 轻量 Node.js 静态服务（内置简单数据接口）

## 本地运行
1. 安装 Node.js（建议 16+）。
2. 在项目根目录执行：
   ```bash
   node server.js
   ```
3. 打开浏览器访问：`http://localhost:3000`

## 目录结构
- `齐大植物大战僵尸.html`：主页面（默认入口）
- `server.js`：静态资源服务与本地数据接口
- `css/`、`js/`：样式与脚本资源
- `data/`：本地数据文件（运行后自动创建）
