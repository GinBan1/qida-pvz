# Contributing

感谢你对本项目的关注！欢迎提交 Issue 和 Pull Request。

## 提交建议

1. 先通过 Issue 描述问题或需求
2. Fork 并创建功能分支
3. 保持改动聚焦、清晰、可复现
4. 提交 PR 时说明背景、改动内容和验证方式

## 代码与文档约定

- 保持现有目录结构与命名风格
- 不提交无关格式化改动
- 若修改功能，请同步更新相关文档

## 本地验证

目前项目为轻量结构，建议至少完成：

- `node server.js` 可正常启动
- 页面可正常打开并完成基础游戏流程
- 排行榜/昵称接口可正常读写

## Pull Request 建议内容

- 改动摘要
- 影响范围
- 手动测试步骤
- 如有 UI 变化，附截图或录屏
